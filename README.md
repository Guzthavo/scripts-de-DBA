 Curso de Banco de Dados Avançado (BDA) 

Repositório criado para consolidação dos aprendizados técnicos desenvolvidos durante as aulas de Banco de Dados e Sistemas Embarcados (IoT).

Tecnologias e Ferramentas Utilizadas

- **SGBD:** MySQL
- **Linguagens:** SQL (DDL, DML, DQL), C++ (Arduino Framework)
- **Hardwares:** ESP32-S3 N16R8
- **Ferramentas de Modelagem:** brModelo, AWS Cloud Documentation
- **Ambiente de Desenvolvimento IoT:** PlatformIO / VS Code

Estrutura e Engenharia de Bancos de Dados

O projeto abordou a criação de esquemas relacionais, aplicação de restrições de chaves estrangeiras (`FOREIGN KEY`) com ações em cascata (`ON DELETE CASCADE`), e a normalização de dados através das **Formas Normais (1FN, 2FN, 3FN)**.

1. **Escola Code:** Gerenciamento de trilhas e tags de cursos.
2. **Senai Play:** Catálogo expansivo com mais de 100 registros cruzando áreas tecnológicas e níveis de cursos utilizando junções complexas (`INNER JOIN`).
3. **Serviços de Elétrica:** Arquitetura para controle de manutenção e faturamento por níveis de tensão.


Desenvolvido como critério de fixação de estudos práticos.
